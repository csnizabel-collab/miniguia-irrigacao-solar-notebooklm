# miniguia-irrigacao-solar-notebooklm
Guia de aprendizado sobre as possibilidades da energia fotovoltaica off-grid em sistemas de irrigação. Utilizando o NotebookLM, o projeto explora como a IA auxilia na compreensão de tecnologias limpas para o agro, abordando desde o funcionamento básico até a viabilidade sustentável em áreas isoladas.
